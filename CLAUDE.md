# CLAUDE.md — Контекст проекта для агента

## Проект

**Ubik** — открытая децентрализованная платформа для распространения образовательного контента.
Три формата: рилсы (видео до 3 мин), голосовые заметки (аудио до 5 мин), слайды (до 7 карточек).
Федеративная архитектура: независимые Go-ноды обмениваются контентом по открытому протоколу.
